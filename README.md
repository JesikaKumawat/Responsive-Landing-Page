# Responsive-Landing-Page
A simple and responsive landing page built using **HTML and CSS**.
This page is designed to promote a mobile app idea and encourage users to start a free trial.
---
# Features
- Clean and modern UI
- "Launch Your App Idea Today" hero section
- Simple CTA (Start Free Trial)
- App promotion banner with illustration
- Sections included: Features, Pricing and Contact
---
## Technologies Used
- **HTML**
- **CSS**
---
## How to Use
1. Download or clone the project folder
2. open the 'index.html' file in any browser
3. Customize text, colors, or images from 'style.css' and 'index.html'
